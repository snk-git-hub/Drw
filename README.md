## Development

### Prerequisites
- Node.js
- pnpm

### Setup
```sh
npx create-turbo@latest
pnpm install
```

### Commands
- `pnpm dev`: Start development servers
- `pnpm build`: Build all packages/apps
- `pnpm lint`: Run linter
